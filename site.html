Protocolos de Rede

HTTP

+

O HTTP (HyperText Transfer Protocol) é o protocolo principal para transferência de páginas web entre clientes (navegadores) e servidores. Sua versão segura, HTTPS, utiliza TLS/SSL para criptografar a comunicação.

SMTP

+

O SMTP (Simple Mail Transfer Protocol) é responsável pelo envio de e-mails entre clientes e servidores de correio e também entre servidores de e-mail.

IMAP

+

O IMAP (Internet Message Access Protocol) permite acessar e gerenciar e-mails diretamente no servidor, mantendo a sincronização entre múltiplos dispositivos.

DNS

+

O DNS (Domain Name System) traduz nomes de domínio legíveis (como www.exemplo.com) em endereços IP, permitindo a localização de servidores na Internet.

IP

+

O IP (Internet Protocol) gerencia o endereçamento e roteamento de pacotes de dados na rede. Cada dispositivo possui um endereço IP único (IPv4 ou IPv6).

P2P

+

O modelo Peer-to-Peer (P2P) permite comunicação direta entre dispositivos sem servidor central. Muito usado em redes de compartilhamento, como BitTorrent.
